How It Works

Simple steps to use the Customer Log module in POS.

Step 1: Install the Module

Install the “Customer Log” module from your Odoo Apps.

Step 2: Access Customer History

Select a customer in POS to view their order history and purchase totals.

Step 3: Reprint Invoices

Reprint invoices directly from the customer log for quick reference.
